Данный проект демонстрирует сохранение объекта Java на диск и чтение
с диска этого объекта. 
При чем объект имеет предка, и все манипуляции проводятся с объектом,
приводя его к предку 
(создаем Child, получаем ссылку на Parent из этого Child, записываем Parent;  
  читаем объект как Parent, приводим объект к Child)

объект должен нормально прочитаться.
