Пакет, который содержит необходимые маркерные интерфейсы для автоматизации процесса работы с JMX:
  • IMBeanAttribute - интерфейс для реализации в JavaBean объекте, при чем все property JavaBean объекта будут считаться аттрибутами (необходимо иметь get/set методы)
  • IMBeanOperations - интерфейс для реализации в объекте, в котором все методы будут доступны из вне для вызова (могут возвращать значения)
  • IMBeanNotification - необходимо создать интерфейс, который наследует данный, имеет методы только без возвращаемого типа и только с одним параметром, затем необходимо создать реализацию данного интерфейса, можно пустую, и передать созданный объект в NotificationAware, из которого затем получить proxy-объект, реализующий все методы для передачи в JMX как Notification
