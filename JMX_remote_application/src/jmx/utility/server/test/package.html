Пример работы с JmxUtility:
  • TestAttribute - класс, реализующий маркерный интерфейс IMBeanAttributes, это JavaBean у которого все get/set методы могут удаленно вызываться с помощью JMX
  • TestOperations - класс, реализующий маркерный интерфейс IMBeanOperations все методы которого доступны для удаленных вызовов через JMX
  • ITestNotification - интерфейс, реализующий маркерный интерфейс INotification, пустая реализация данного интерфейса (только логгирование)- TestNotification, передается в качестве параметра в NotificationAware, а после построения JmxUtility вынимаем оттуда (из NotificationAware) proxy-object ITestNotification
  • TestJmxUtility - объединение всех компонентов вместе
