Данный проект демонстрирует возможность ввода данных с командной строки,
и интерпретирование Java этих данных, 
например при вводе строки:
"this is temp" "and temp again"    and     not    run

тело main(String[] args) получит массив из строк:

1 : this is temp count:3
2 : and temp again count:3
3 : and count:1
4 : not count:1
5 : run count:1


 первая цифра - порядковый номер 
 последняя цифра - кол-во блоков букв(слов), разделенных пробелами 
 
 Вывод:
 если  заключать входные параметры в двойные кавычки, то последовательность символов будет восприниматься как
 один элемент 
 при чем все элементы(слова), которые разделены пробелами будут восприниматься без начальных и конечных пробелов, 
 то есть будут восприниматься как строки, над которыми была произведена операция trim()
