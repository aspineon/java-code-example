Данный проект предназначен для сохранения файлов XML, сгенерированных SVN
( пример получения фала XML из SVN: svn log -r17760:17766 -v --xml >> svn_log.xml),
в базу данных

Пример аргументов для запуска данного приложения:

  • -svn_xml_file /tmp/temp/svn_log.xml
  • -db_url "jdbc:oracle:thin:@(DESCRIPTION =(ADDRESS_LIST =(LOAD_BALANCE=OFF)(FAILOVER=ON)(ADDRESS = (PROTOCOL = TCP)(HOST = 10.91.57.225)(PORT = 1521))(ADDRESS = (PROTOCOL = TCP)(HOST = 10.91.57.227)(PORT = 1521)))(CONNECT_DATA =(SERVICE_NAME = rtn7uadg.world)(FAILOVER_MODE=(TYPE=SESSION)(METHOD=BASIC)(RETRIES=180)(DELAY=5))))"
  • -db_login xxxxx
  • -db_password yyyyy
  • -log_level error

Таблицы в базе данных для сохранения:
SVN_LOG_ENTRY
ID unique identifier
REVISION svn revision
AUTHOR author of SVN revision
DATE_WRITE time of commit
MESSAGE comment of commit
TASK number of Task from JIRA

SVN_LOG_ENTRY_FILE
ID unique identifier
ID_SVN_LOG_ENTRY reference to SVN_LOG_ENTRY
KIND attribute of element
ACTION Modified, Append, Delete - type of file manipulation
SVN_PATH path to file in SVN
